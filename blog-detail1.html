Jarvis System

This project was to integrate Arduino and laptop in order to build a Smart Assistant at home. I call it Jarvis in Iron man. There’s an Arduino-like open source firmware to help us prototype Iot products : Nodemcu. As an Iot device, it is less than $2 WI-FI MCU ESP8266 integrated and easy to prototyping development kit. And I need a laptop as a central management system. It uses several techniques including speech recognition, web crawler, socket, sqlite, Messenger API written in Python and Nodejs.

Ask weather & news

i applied crontab to grap weather and news information every three hours. First write an web crawler program and set up cron job using crontab. Crontab is a time-based job scheduler in Unix-like computer operating systems. In the program, I need to store data as JSON file, and everytime I call Jarvis for weather and news, Jarvis reads the corresponding data and shows the information.
