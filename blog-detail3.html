YOLO V2 Training

This is a starter implementation of YOLOv2 model for object classification and detection using a pretrained weight. My Goal is to explain how YOLO algorithm works.

Red Blood Cell datasets

Reference: Datasets download
In RBC_dataset directory, 366 local annotations(.xml) are stored in Annotations directory, and coresponding images(.jpg) are in JPECImages directory.

Environment

Ubuntu 16.04.3 LTS
Anaconda3-4.5.11
Cuda v7.5.17
Tesla K40c


parse_annotation

In preprocessing.py, parse_annotation function is to transfer all xmls into a list, and then output 'all_imgs' and 'seen_labels' to count objects in all classes.


BatchGenerator

BatchGenerator is a function to suit all YOLO augmentation needs. Its output structure is a multi-dimensional array as shown below. 'x_batch' includes train (RGB) images, and 'y_batch' is the ground truth for calculating loss value. The one-hot encoded 'class' in (x, y, h, w, conf, class) can also express as (x, y, h, w,conf, class0, class1, class2,...) based on how many classes in datasets.
